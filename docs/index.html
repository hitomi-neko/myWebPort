神奈川県茅ヶ崎市出身。2012年、平塚中等教育学校入学。5年次で重度の起立性調節障害を発症。1年半ほど不登校状態となるも、 2018年に卒業。同年、東京都立大学に入学。病状は快方に向かったものの、 不登校時に抱いた教育への疑問や学校への不信感から、教育学への関心が高まる。
また、自分と同じような思いをする子どもを減らしたいとの思いから、子どもに関するボランティア活動にも積極的に取り組む。 そうした経緯で同大学のボランティアセンターに学生コーディネーターとして所属し、ボランティア活動を通して学生の学びをサポートする活動をしている。
現在は学校現場におけるデザインの利活用や、子どもたちを取り巻く問題、社会教育・生涯学習についてをテーマに学んでいる。
