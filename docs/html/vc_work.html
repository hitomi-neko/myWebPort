ボランティアセンターでの制作物です。
私が大学1年次から所属しているボランティアセンターではポスター・チラシの制作に加え、自主企画による制作も行っている。

都立大ボランティアセンター看板
2020/02~2020/03

使用ツール：Illustrator
ボランティアセンターでの制作（デザイン）

大学名が首都大から都立大へと変化する中で、センターの看板も作りなすこととなり、そのデザインを担当した。 制作にあたっては複数の色パターンを制作し、そこから職員の方々の会議で色が決定した。

ボランティアによる人と人との繋がりをテーマにイラストを制作した。 また、ボランティアセンターとは「人と人」「ボランティアと人」「人と地域」…と多様な繋がりを結ぶ機関である。 そうしたイメージをリボンや一筆書きという絵に込めた。 そして都立大のスクールカラーである青を多く使ったデザインに仕上げた。

ぼらせん便り
2018/11~

使用ツール：Illustrator
ボランティアセンターでの制作（担当：デザイン・編集）

「学生から学生に」をコンセプトに制作した広報誌。 ボランティア団体で活動する学生の「生の声」を届けることを目的とした企画。 ボランティアと聞くと、「真面目な人がやっている」「すごいこと」などの固定観念を抱きがちである。 そうした固定観念に対して、「ボランティアはあなたと同じような普通の学生がやっている」と伝えることができれば何か変わるのではないかという思いで制作を開始。
手に取り、軽く読めるように1枚の紙で制作している。

全体的にポップな印象の広報誌。 制作にあたっては、事前に学内のボランティア団体や学生コーディネーターの学生に声をかけ、文章と写真をいただき、 そこから私が編集する。現在、18号まで発行されている。
