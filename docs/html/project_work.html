みやこ祭り「Projection Mapping 2019」
2019/04~2019/11

使用ツール：After Effects,Premire Pro
チームプロジェクト(役割：2019年度代表)

都立大大学祭実行委員会とインダストリアート学科有志の学生とのコラボ企画。 大学祭という、地域の方が多くいらっしゃる場だからこその表現や、この学科を知ってもらうにはどうしたら良いかと 考え、4月からチームで議論を重ねて制作した。 2019年度のテーマは「Re:」。平成から令和へ、首都大から都立大へと変化していく時の流れの中で繰り返されるものも存在する。 そうした「時」や「繰り返し」をテーマに制作した。

メンバーは主にインダストリアルアート学科の1，2年生で結成され、2019年度は20名ほどの学生が参加した。
この企画は作品を制作するだけでなく、学年を超えた学びの場としての機能も存在し、 私を含めた2年生はそうした場づくりも行った。

いきものみっけ！　プロジェクト
2019/07~2019/08

使用ツール：Illustrator
チームプロジェクト(役割：代表、企画、制作)

生物系の学生・アート系の学生を中心とし、有志で行った企画。
「わくわくで、自然や生物に興味をもっと持ってもらおう！」をコンセプトに大学近郊の小学校の教室を一室お借りして行った。 教室内に名前の書かれた生き物の写真を隠し、入り口で渡されたカードに書かれた写真の生き物を探すというゲーム形式で 学校の周りの自然に興味を持ってもらおうという企画。
当日は2～30名ほどの小学生が参加してくれた。わくわくから始まる学びというテーマをいかに実現するか難しいところであったが、 大変勉強になった。
